# MVP 验收标准

1. 多用户与权限
- 可注册至少两个账号并独立登录。
- 两个账号的棋局与统计互不串数据。
- 家长账号可查看孩子进度页；孩子账号不可查看家长详情页。

2. 对局与复盘
- 用户可完成一盘棋录入并回放。
- 系统可输出至少 3 个关键改进点（失误类型 + 建议走法）。

3. 训练闭环
- 系统可基于用户漏洞生成每日训练任务。
- 用户完成训练后，进度看板可反映完成情况与趋势变化。

4. 段位体系
- 系统可展示用户当前段位与升级进度。
- 系统可展示段位评分维度说明与当前短板项。

5. 可用性
- 核心流程可在桌面端完整走通：登录 -> 对局 -> 复盘 -> 训练 -> 看板。
