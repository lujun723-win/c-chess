# MVP 方案（v1）

## 核心闭环
对局记录 -> 复盘分析 -> 漏洞画像 -> 定向训练 -> 进步看板

## 功能清单
1. 账号与家庭组
   - 用户注册/登录
   - 家庭组创建 + 邀请码加入
   - 家长/孩子角色权限
2. 棋局与复盘
   - 棋盘录入、回放
   - 关键步与失误步标注
   - 候选走法与简要解释
3. 训练与进度
   - 每日训练任务（按漏洞类型推荐）
   - 完成记录
4. 看板
   - 失误率、训练完成率、复盘完成率
   - 支持家庭组内按成员切换查看（权限受控）
5. 等级体系（新增）
   - 段位等级（入门/业余段位）
   - 升级进度与缺口提示
   - 评分维度解释（实战/复盘/训练）
   - 六维雷达图（攻击/计算/子力/防守/对抗/成长）

## 版本边界
- v1 只做基础分析能力和基础训练推荐。
- 复杂 AI 教练对话与高精度引擎评估放到 v2。
