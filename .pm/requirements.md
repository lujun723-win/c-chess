# 中国象棋学习网站 - 需求说明（已确认版）

## 背景痛点
- 用户会下中国象棋基本规则，但不愿背谱，缺少体系化训练方法。
- 实战常出现“走完才发现漏洞”，容易进入对手陷阱。
- 长期缺少复盘方法，无法稳定总结漏洞与进步路径。

## 目标用户
- 业余象棋学习者（懂规则、长期实战、缺系统提升方法）。
- 家庭用户：至少支持家长与孩子共同使用。

## 产品目标
1. 降低实战中的关键漏算与陷阱失误。
2. 提供可操作的复盘建议，而非泛泛评价。
3. 建立“个人漏洞画像 -> 定向训练 -> 效果验证”的闭环。
4. 支持多用户账户体系，个人数据强绑定账户。
5. 建立可解释的成长评估体系（段位/等级），让进步可视化。

## 关键需求（确认）
1. 多用户系统：支持至少两名用户同时使用（你和孩子）。
2. 账号绑定：棋局、复盘、训练、统计全部与账号绑定。
3. 数据隔离：默认仅可访问本人数据。
4. 家庭组机制：支持创建家庭组并邀请成员。
5. 家长视角：家长可查看孩子学习进度与漏洞画像；孩子默认不可查看家长详细数据。
6. 成长评估体系：基于实战、复盘、训练三类指标计算等级与升级进度。

## 非目标（MVP不做）
- 社区、直播、评论区
- 排位赛/匹配系统
- 大而全开局百科
